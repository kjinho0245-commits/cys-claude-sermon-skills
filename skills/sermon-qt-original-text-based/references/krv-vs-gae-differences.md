# 개역한글(KRV)과 개역개정(GAE) 토씨 차이 빈출 사례

LLM이 학습한 한국 성경 데이터는 *개역한글*(KRV, 1961)이 압도적으로 많아, 사용자가 "개역개정"(GAE, 1998)을 지정해도 자동으로 개역한글 토씨를 옮기는 위험이 매우 크다. 본 자료는 큐티 산출 시 자주 혼선되는 구절을 정리한다.

## 자주 큐티에 등장하는 본문의 차이 사례

### 시편 23:1
- KRV: "여호와는 나의 목자시니 **내가** 부족함이 없으리로다"
- GAE: "여호와는 나의 목자시니 **내게** 부족함이 없으리로다"
- 차이: "내가" → "내게"

### 시편 23:2
- KRV: "그가 나를 푸른 초장에 누이시며 쉴 만한 물 가으로 인도하시는도다"
- GAE: "그가 나를 푸른 풀밭에 누이시며 쉴 만한 물 가로 인도하시는도다"
- 차이: "초장" → "풀밭", "물 가으로" → "물 가로"

### 시편 23:4

2. 위 빈출 패턴 중 산출물에 들어간 인용이 KRV 흔적인지 GAE 흔적인지 자가 점검.
