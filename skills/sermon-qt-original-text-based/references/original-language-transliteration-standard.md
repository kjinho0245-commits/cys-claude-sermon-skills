# 헬라어·히브리어 한글 표기 표준

큐티 산출물에 원어 단어를 인용할 때 적용하는 한국 신학계 통용 표기 표준. SKILL.md의 *원어 한글 표기 표준* 절을 보강한다.

## 1. 헬라어 — 학문적 Erasmian 표기

한국 신학교(장로회신학대학교·총신대학교·고신대학교·합동신학대학원·웨스트민스터·풀러 한인 등)에서 통용되는 *학문적 Erasmian* 발음을 한글로 옮긴다. modern Greek 발음(예: η→이, θ→쓰, ου→우-)이 아닌, 16세기 에라스무스가 재구한 고전·코이네 발음 체계.

|--------|-----------|------|------|
| β | b | ㅂ | |
| γ | g | ㄱ | γγ → "ㅇㄱ", γκ → "ㅇㅋ", γξ → "ㅇㅅ", γχ → "ㅇㅋ" |
| δ | d | ㄷ | |
| ζ | z | ㅈ (어두) / ㅅㅈ (어중) | 예: ζωή → 조에 |
| θ | th | ㅌ (강하게) | "쓰"는 modern, "트"가 Erasmian |
| κ | k | ㅋ | |
| λ | l | ㄹ | |
| μ | m | ㅁ | |
| ν | n | ㄴ | |
| ξ | x | ㅋㅅ | |
| π | p | ㅍ | |
| ρ | r | ㄹ | 어두 ῥ → "ㄹ" (거친 숨표 무시) |
| σ / ς | s | ㅅ | |
| τ | t | ㅌ | |
| φ | ph | ㅍ | |
| χ | ch | ㅋ (강하게) | |

| 부드러운 숨표 ̓ | (없음) | 표기 없음 | 예: ἀγάπη → 아가페 |

## 3. 표기 결정 우선순위 (충돌 시)

1. **개역개정**이 단어를 한글로 표기한 경우(예: "여호와", "엘로힘", "샬롬"), 개역개정 표기를 따른다.
2. 개역개정에 없으면, 한국 신학계 표준 사전·교과서 표기를 따른다.
3. 학자별 표기가 갈리는 경우, 가장 일반적인 표기 한 가지를 골라 산출 내내 일관 사용.
4. 출력에는 한 가지 표기만 사용한다. "샬롬 또는 샬롬" 같은 혼선 금지.

## 4. 자주 발생하는 표기 오류와 교정

| 오류 표기 | 정정 표기 | 사유 |
|-----------|----------|------|
| ζωή → "조이" | "조에" | η는 Erasmian에서 "에"이지 "이"가 아님 |
| πίστις → "피스티" | "피스티스" | -ις 어말 ς 누락 |
| πνεῦμα → "프뉴마" | 그대로 | 정상 |
| εὐαγγέλιον → "에반겔리온" | "유앙겔리온" 또는 "에우앙겔리온" | εὐ 표기 |
| κύριος → "큐리오스" | "퀴리오스" | υ는 "위/우", "유"가 아님 |
| χάρις → "차리스" | "카리스" | χ는 "ㅋ"(강), "차"가 아님 |
| θεός → "테오스" | 그대로 | 정상 ("쎄오스"는 modern Greek) |
| διαλέγομαι → "디아레고마이" | "디알레고마이" | λ 위치 |
| חֶסֶד → "헤세드" | 그대로 | 정상 |
| שָׁלוֹם → "샬롬" | 그대로 | 정상 (개역개정 표기 채택) |
| יהוה → "야웨" | "야훼" 또는 "여호와" | 개역개정은 "여호와" |
