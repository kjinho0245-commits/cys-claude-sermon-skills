# 할루시네이션 방지 체크리스트

## 매칭 주장 전 내부 점검 프로토콜 (3-Gate)

### Gate 1: 유물/문헌 실존 확인

매칭에 사용할 유물 또는 고대 문헌에 대해:

- [ ] 박물관명이 실존하는 기관인가?
- [ ] 카탈로그 번호가 구체적으로 알려져 있는가?
- [ ] 발견 연도와 발굴자가 알려져 있는가?
- [ ] 이 유물이 해당 박물관 공식 컬렉션에 등재되어 있는가?

**실패 시 처리**: 카탈로그 번호 없이 "소장처만 명시" 또는 "[번호 확인 필요]" 표기. 번호를 지어내지 않는다.

### Gate 2: 학자/논문 실존 확인

인용할 학술 자료에 대해:

- [ ] 저자가 실존하는 학자인가? (이름이 구체적으로 알려진 인물)
- [ ] 논문/저서 제목이 구체적으로 알려져 있는가?
- [ ] 출판연도와 출판사가 구체적으로 알려져 있는가?
- [ ] 학술지명이 실존하는 학술지인가?

**실패 시 처리**: "학자들에 따르면" 식의 모호한 표현 금지. 구체 인용이 불가능하면 "[구체 출처 확인 필요]" 표기.

### Gate 3: 학계 컨센서스 여부 확인

제시할 주장에 대해:

- [ ] 이것이 주류 학자들의 견해인가, 아니면 소수/비주류 견해인가?
- [ ] 이 동일시(identification)에 학계 논쟁이 있는가?
- [ ] 논쟁이 있다면 주류 측과 소수 측을 균형 있게 제시했는가?

**실패 시 처리**: 논쟁 중인 사항에 "[DEBATED]" 표기. 소수설을 주류설처럼 제시 금지.

---

## 절대 금지 목록

| 금지 행동 | 대안 |
|----------|------|
| 없는 카탈로그 번호 창작 | "[번호 미확인]" 또는 알려진 번호만 표기 |
| 없는 학자 이름 창작 | 알려진 학자만 인용, 불확실하면 생략 |
| "대부분의 학자들이..." 모호한 표현 | 구체 학자 이름 + 논문 명시 |
| 소수설을 컨센서스로 제시 | 학계 내 위상 명시 (주류/소수) |
| "증명한다"는 표현 | "일치한다", "지지한다", "확인한다"로 대체 |
| 연대 단정 (논쟁 중인 경우) | "기원전 X세기경", "학계 논쟁 중" 표기 |
| 지명 동일시 단정 | "현재의 X로 비정됨", "동일시 논쟁 중" 표기 |

## 경계선 케이스: 이것은 제시할 수 없음

### 출애굽의 파라오 동일시
- 람세스 2세(기원전 13세기설) vs 투트모세 3세(기원전 15세기설) 논쟁
- 학계 미합의. "출애굽기의 파라오는 람세스 2세" 단정 불가
- 허용 표현: "출애굽 연대는 기원전 13세기 또는 15세기로 학자들 간 논쟁 중이며, 어느 파라오인지 특정되지 않음"

### 노아 방주의 위치
- 고고학·지질학 근거 없음. 제시 불가

### 요나와 역사적 니네베 멸망
- 요나서의 40일 회개 이야기는 역사 기록과 직접 연결 불가. 제시 불가

### 솔로몬 광산 = 팀나 구리 광산
- 상관관계 있으나 "솔로몬의 광산"임을 증명하는 직접 증거 없음. [DEBATED]

## 신뢰도 등급 사용 기준

| 등급 | 조건 | 예시 |
|-----|------|------|
| `[CONFIRMED]` | 물리적 유물 + 학계 주류 합의 | 빌라도 비석, 산헤립 프리즘 |
| `[PROBABLE]` | 간접 증거 + 학자 다수 지지 | 고레스 실린더-에스라 연결 |
| `[DEBATED]` | 증거 존재하나 해석 논쟁 | 텔단 비석 일부 해석 |
| `[UNCERTAIN]` | 증거 불충분, 추론 단계 | 솔로몬 성전 정확한 위치 |
| `[OUT OF SCOPE]` | 역사학·고고학 영역 밖 | 창조, 노아 홍수의 우주적 규모 |
